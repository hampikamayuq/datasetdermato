# Source Notes

## DermNetNZ

Use DermNetNZ as a source candidate with explicit license review. A page URL and image URL should be recorded before any file is downloaded or transformed.

Recommended manifest fields are in `metadata/sources/dermnetnz_manifest.csv`.

## Skinive dataset review

The Skinive article is useful as a catalog of candidate public datasets. It is not itself a dataset source for images. The listed datasets include ISIC Archive, HAM10000, DermaMNIST, SD-198, PAD-UFES-20, PH2, Derm7pt, Fitzpatrick 17K, BCN20000, and SIIM-ISIC.

Each dataset needs separate license and access review before ingestion.

## ISIC

The included ISIC files in `references/` are used as inspiration for field names, validation rules, and de-identification behavior. The Santa Casa project can keep richer project-specific fields whenever scientifically useful.
